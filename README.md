Copy Twitter friends/favorites to another account
=================================================

```
% cd twitter-move
% . install.source
% ./getfriends.py
% ./addfriends.py
% ./getfavs.py
% ./addfavs.py
```

Please be aware of the Twitter API rate limits. The above process may take several days to complete if you have a lot of friends and favorites.
